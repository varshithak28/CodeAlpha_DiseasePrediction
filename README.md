# Heart Disease Prediction using Machine Learning

## Project Overview

This project predicts the likelihood of heart disease using patient health data using Machine Learning techniques.

## Dataset

Heart Disease Dataset containing 1025 patient records.

## Algorithm Used

* Logistic Regression

## Accuracy

* 79.5%

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Project Outcome

The model was trained and tested successfully and achieved an accuracy of approximately 79.5% in predicting heart disease.
